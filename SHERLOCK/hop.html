How To Play

SHERLOCK is an object finding game made by Irish.K.Sajan 2020BCS0004.The game screen consists of 3 parts:
1. A picture from which objects are to be found
2. List of Objects to find
3. A table showing the Time taken, The score and The Time limit
When one finds an object given the list and clicks on it he/she will be awarded 5 points.If one is able to complete a the one level in the prescribed maximum time 
He/She will be able to attempt the next level. If one fails in any one level he/she will have to start again from the first level. There is a total of 6 levels
